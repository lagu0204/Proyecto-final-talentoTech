Proyecto 2

El proyecto de inventario es una aplicación sencilla diseñada para gestionar productos de forma básica. Las funcionalidades principales incluyen la posibilidad de agregar productos, registrar su nombre, descripción, precio y stock disponible. La aplicación está desarrollada con Angular para el frontend, proporcionando una interfaz interactiva y fácil de usar, mientras que utiliza MySQL Workbench como herramienta de gestión de la base de datos para almacenar y organizar los datos de los productos. Es ideal para pequeños negocios o proyectos personales que requieren un control básico de inventarios.
